《通信软件开发与应用》

实验内容

该网站是一个介绍重庆的网站,主要包含了重庆的历史文化，景区，美食等相关介绍。


开发过程

采用html+css开发静态页面，借用bootstrap完成首页轮播图的展示


开发问题

问题1:css样式未生效，css样式不符合预期等。

分析:css未生效可能是语法错误或层叠样式优先级的问题.

解决:检测语法格式，是否引入css文件，提高选择器的优先级等.


问题2:使用浮动布局时，页面发生较大变动

分析:因为浮动的元素会脱离原来的常规流。浮动元素可能会覆盖一些非浮动的元素

解决方法:使用clear属性来清除


问题3:定位相关问题

暂未解决


技术总结

DIV+CSS的布局方法简单来说就是使用div标签作为容器，使用CSS技术来排布div标签的布局方法。

1.CSS总结:

首先是css的选择器可以大致分为5类，简单选择器(根据名称、id、类来选取元素) ，组合器选择器(根据它们之间的特定关系来选取元素) ，伪类选择器(根据特定状态选取元素) ，伪元素选择器(选取元素的一部分并设置其样式) ，属性选择器(根据属性或属性值来选取元素) ID选择器和类选择器是最常用的选择器。 其次是选择器的优先级: !important inline style(内联) id selector (id) class selector (类) element selector (标签) * universal selector (全局)

2.DIV总结:

div盒子模型包括border(边框)，padding(内边距)，margin(外边距)

3.块级元素与行内元素:

所有的XHTML页面元素只有两种，一是块级元素一是行内元素。块级元素一般都从新行开始，它可以容纳行内元素和块级元素。行内元素只能容纳文本或其他行内元素。
